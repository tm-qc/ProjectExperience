# 開発・実装内容

【概要】  
ECサイトの新規開発プロジェクトに参画。
市場調査から設計・実装・テストまで担当し、ECサイト構築および関連API開発を実施した。

【どのような機能の開発・実装か】
- ECサイト基盤構築
- API開発
- Bagisto(e-commerce)分析、連携
- 画面開発
- ドキュメント作成

【課題・問題点】  
新規サービス開発のため、競合サービスの調査や市場分析を行いながら、
要件整理およびシステム構築を進める必要があった。

【打ち手・使用した技術】

■チーム対応  
- Laravel 11を利用したバックエンド開発
- Next.jsを利用したフロントエンド開発
- MySQLによるデータ管理
- Tailwind CSSを利用したUI実装
- Dockerによる開発環境構築
- GitHubによるソースコード管理
- Bagisto(e-commerce)を利用したECサイト構築
- 市場調査および競合ECサービス分析を実施
- 上層と必要な機能のすり合わせ、要件整理
- Bagisto採用中止、参考にLaravelで必要機能を実装する方針に変更

■個人対応  
- CTOと協議し、保守性および将来的な運用面を考慮してBagistoの採用を見送り、Laravelによる独自実装へ方針変更
  (細かいバグ報告、バージョンアップに寄るバグの解消が困難、公式の回答から対応時期が未定 etc)
- GitHub Wikiを利用したドキュメント管理基盤の整備
- GitHub Projectsを利用した案件管理基盤の整備


 【思考プロセス】  
過去の経験から初動をできる範囲で整えていくことが大事だと思っていたので、PJの始動時期だったこともあり、
案件、ドキュメント、開発方針の共有は特に意識して作業を行っていました。


【成果】
- Bagisto(e-commerce)サービスとAPI連携したECサイト基盤を構築
- e-commerceを参考に必要機能を実装
- GitHub Wikiを利用したドキュメント管理環境を整備
- GitHub Projectsを利用した案件管理基盤を整備
- 市場調査から要件整理、設計、実装まで担当
