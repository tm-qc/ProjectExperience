# 開発・実装内容

【概要】  
森林組合向け業務管理アプリ開発に向けた技術調査およびプロトタイプ作成を担当。  
黒板撮影、画像合成、位置情報取得、文字入力、Firebaseへのアップロード機能を備えたアプリの実現可能性を検証した。

【どのような機能の開発・実装か】
- 黒板撮影機能
- 画像合成機能
- 位置情報取得機能
- 文字入力機能
- Firebaseアップロード機能
- プロトタイプ作成

【課題・問題点】  
新規サービス開発のため開発方式や技術選定が未確定であり、要件を満たす開発基盤の調査および検証が必要だった。

【打ち手・使用した技術】

■チーム対応  
※個人調査、サンプル開発なのでなし

■個人対応  
- .NET MAUIの調査、独習およびサンプルアプリ作成
- Flutter（3.29.1）およびDartの調査、独習およびサンプルアプリ作成
- Firebaseの調査および接続検証
- 各技術で実現可能な機能や開発効率を比較検証

 【思考プロセス】  
 社内、私自身もあまり知見のない領域だったため、開発のしやすさも大事ですが、
 一般的な事例、シェア率、情報量の多さを優先し選定することを意識していました。

【成果】
- 実現予定機能（黒板撮影、画像合成、位置情報取得、Firebase連携）の実装可否を検証
- .NET MAUIとFlutterを比較検証し、将来的な保守性やシェア率を考慮した技術選定を実施
- .NET MAUIおよびFlutterによるプロトタイプを作成
- Firebaseとの連携方式を調査・検証
