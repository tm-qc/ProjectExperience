# 開発・実装内容

【概要】  
社内および顧客向け業務支援ツールの新規開発・改修プロジェクトに従事。
業務改善を目的としたツール開発を中心に、要件定義から設計・実装・テスト、運用支援まで担当した。

【どのような機能の開発・実装か】
- シフト管理ツール
- チーム進捗管理ツール
- チャット業務支援ツール
- 機器貸出管理ツール
- アンケートツール
- 案件管理ツール
- ナレッジ管理サイト改修
- フローチャート作成ツール改修

【課題・問題点】  
各部門でExcelや手作業による管理が多く、業務効率化および情報共有の改善が求められていた。
また、長期間運用されているシステムでは仕様が属人化しているケースもあり、改修や保守を行うために調査・整理が必要だった。


【打ち手・使用した技術】

■チーム対応  
- PHP5を利用したWebアプリケーション開発
- MySQLによるデータ管理
- JavaScript / jQueryによる画面開発
- Vue.js / Vuetifyを利用したレガシーシステムの改善

■個人対応  
- エンドユーザーとの要件整理および仕様調整
- 工数見積、進捗管理、作業割り振り
- シフト管理ツールの新規開発
- チーム進捗管理ツールの新規開発
- 既存システムのリファクタリングおよびVue.js化
- マニュアル作成および新人教育

【思考プロセス】  
数々のツールをチーム要件に応じてヒアリング、設計するにあたり、開発の事がわからない担当も多く、
相互納得できる状態になるように双方合意を特に意識して対応してきた。

また仕様書なし、作成者不在のツールもあり、改修するにあたり、
改修後の自身での動作確認、利用チームのクライアントに協力を仰ぎ、極力致命的なバグが起きないように留意した。

またこの作業の過程で仕様書を作成するようにチームに促し、今後の安全も図った。

【成果】
- 複数の業務支援ツールを開発し業務効率化に貢献
- 要件定義から設計、開発、テストまで一貫して担当
- プロジェクトリーダーとして進捗管理およびメンバー管理を実施
- レガシーシステムの改善および保守性向上に貢献
- エンドユーザーとの調整を行い、運用に即したシステム改善を実施
