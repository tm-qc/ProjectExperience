# 開発・実装内容

【概要】  
森林組合でアナログ管理されていた顧客情報や案件情報をWebツールでデジタル化し、一元管理するシステムを開発。

【どのような機能の開発・実装か】
- 顧客管理
- 案件管理

【課題・問題点】  
顧客要望をシステムとして実現するため、業務要件を設計へ落とし込む必要があった。

【打ち手・使用した技術】

■チーム対応  

- ASP.NET Core (.NET 8) を用いたWebアプリケーション開発
- MySQLを利用したデータ管理
- HTML / CSS / JavaScriptによる画面作成
- GitHubによるソースコード管理

■個人対応  

複雑な業務要件だったため、PMの設計書や初回の顧客折衝だけでは、イメージが追い付かず、詰め切れない部分があった。
そのため、実際に動くところまで作成し、PMに動作が想定通りか確認しながら作成を行った。

PMはその後顧客に確認し、修正があればまたエンジニア側で確認し修正という流れだった

 【思考プロセス】  
複雑な業務内容だったため、「設計書→実装→動作確認→PMの想定通りか確認」を行い、
相互で認識相違がないように意識していました。


【成果】  
顧客要件に沿った業務管理Webツールの開発を実施し、設計から実装・テスト、修正まで一貫して担当した。
