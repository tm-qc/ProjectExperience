# 開発・実装内容

【概要】
自社開発CMSの保守・改修プロジェクトに参画。
クライアントからの依頼対応や不具合修正、新規導入時の動作確認などを担当した。

【どのような機能の開発・実装か】
- ヘッダー、フッターのカスタマイズ
- 画像一括更新・削除機能の改修
- 新規クライアント向けCMS導入支援
- SSL設定
- 動作確認およびリリース対応

【課題・問題点】
既存CMSを利用する複数クライアントからの要望対応や不具合修正を行う必要があった。
また、新規クライアント向けにサーバー環境ごとの動作確認や設定作業が必要だった。

【打ち手・使用した技術】

■チーム対応  
- PHP5 / CodeIgniter2 を利用したCMS保守・改修
- MySQLによるデータ管理
- HTML / CSS / JavaScript / jQueryによる画面改修
- GitHubによるソースコード管理
- CentOS環境での運用対応

■個人対応  
- クライアント要望に応じたヘッダー、フッター改修
- 画像一括更新・削除機能の不具合調査および修正
- 新規サーバー環境でのCMS動作確認
- SSL設定対応
- テストおよびリリース作業
- CMSに関する社内研修の受講

 【思考プロセス】
全体の仕様、影響が読めない環境だったため、改修後の動作確認と簡潔に改修内容を共有し私より経験があるエンジニアに、
予想外の影響が出ないか確認し、慎重に進めるようにしていた


【成果】
- クライアント要望に応じたCMS改修を実施
- 既存機能の不具合修正および動作確認を実施
- 新規クライアント向けCMS導入支援を担当
- 保守・改修からテストを担当
